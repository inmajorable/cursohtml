Introducción a HTML5

HTML5 es el lenguaje de marcado utilizado para crear páginas web. Permite definir la estructura y el contenido de una página web utilizando etiquetas y elementos.

Estructura Básica

Una página web HTML5 consta de una estructura básica que incluye elementos como <html>, <head>, <title>, <meta>, <body>, y más.
